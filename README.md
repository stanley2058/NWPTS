# NTOU Web Programming TA System
*This project is created for the Web Programming class.*

![](NWPTS.png)


## Using
- Angular
- Angular Material
- Angularfire
- Firebase

## Build And Deploy
1. Replace the Firebase project tokens in `app.modules.ts`
2. `ng deploy`

## Run Angular Dev Server
1. `npm i -g @angular/cli`
2. `npm i`
3. `ng serve --open`
